# Netflix Data Pipeline 🎬

This project simulates a real-world data engineering pipeline using:

- AWS (S3 + Glue)
- SQL transformations
- GitHub & CI/CD
- Power BI dashboard

## Project Structure

- data/ → raw & processed datasets
- sql/ → transformation queries
- scripts/ → helper scripts (Python if needed)
- pipeline/ → YAML pipelines
- powerbi/ → reports
- docs/ → documentation
